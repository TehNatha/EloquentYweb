Further Instructions

The book says to create a string and test it with console.log(). When you've got it figured out, instead of logging the string out, return it.

When you have finished all the specifications in the book, modify your function to accept a parameter n that, if passed into your function, will dynamically specify the size of the chessboard to return. Remember you can use the logical OR operator to choose the first item that exists:

var size = n || 8;
If there is no n, n || 8 will default to 8.
